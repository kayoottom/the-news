## 📸 Preview

Interface simples que apresenta artigos com:

* imagem do artigo
* título
* fonte da notícia
* descrição

---

## ⚙️ Como rodar o projeto

```bash
npm install
npm start
```

O projeto abrirá em:

```
http://localhost:3000
```
